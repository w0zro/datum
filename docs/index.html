datum

A colorscheme reverse-engineered from color science. Hue angles are anchored on the Okabe–Ito colorblind-safe set, placed in OKLCH, checked against WCAG 2.2 and APCA contrast, and simulated for color-vision deficiency. Every token carries a hue, but saturation encodes frequency: the sparse reference points run at full chroma, the ubiquitous glue (variables, calls, operators) in quiet pastels of the same hue. The two modes share hue identity but are tuned separately — a dark theme is not an inverted light one.

Read the reasoning: the design whitepaper — the research findings, the derivation pipeline, and every measured number.

if &background ==# 'light'
  let s:blue = ['25', '#0068a4']  " h244 L0.50
else
  let s:blue = ['75', '#66b6f4'] " h244 L0.75
endif

" keywords colored, not bold; operators get the pastel
call s:hi('Keyword', s:blue)
-- chroma tiers: full-chroma reference points,
-- pastel glue (variables, calls, params, operators).
local M = {}

function M.setup(opts)
  local retries = 3
  local ok = retries > 0 and opts ~= nil
  for _, name in ipairs(opts.colors) do
    apply(name, retries - 1)
  end
  return ok
end

return M
